Searchspringconfig Module is used to display the correct site id on the search results page based on the url
[dev, stage or production].  The information is entered in the admin panel found under System->Configuration->
LYONSCG tab->Search Spring Configuration.  You will have to enter the dev, stage and production urls and their
corresponding Searchspring Site Id's.  The Searchspring site id is found under the Searchspring Management console.
