Filter Finder.
This module creates new page with url FridgeFilterFinder.
It does not require any preinstalling. All nessesary attributes and caregories are creating with install script.
Usage:
    -- We have for steps which are attributes
    -- After choosing first step we are getting collection of products with chosen attribute value and getting all
       available variants of the second step from that collection.
    -- After choosing second step we are getting collection of products with chosen 1st and second attributes values
       and getting all available variants of the second step from that collectio end so on.
