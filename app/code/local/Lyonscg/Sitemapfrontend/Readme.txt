This module overrides the Sitemap Tree Category class and adds a check for excluding categories
which excludes categories from sitemap on the frontend, if the "exclude_from_sitemap" field
is set to Yes [which is under catalog->categores->Manage Categories via admin panel]
also overrides the Sitemap Category class and adds a check for excluding products
which excludes products from sitemap on the frontend, if the "exclude_from_sitemap" field
is set to Yes [which is found in the "Meta information" tab when you edit the product]
